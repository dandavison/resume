MongKok

A Modern Onepage Resume / Personal Template (WordPress Version)

Key Features

MongKok is a modern onepage resume / personal WordPress theme on Bootstrap 3 with lots of options for customization. The key features of this theme include:

  • Modern design
  • Responsive layouts
  • Flexible color scheme (single color / dual colors)
  • 5 predefined color schemes
  • 4 navigation bar positions (top, bottom, left, right)
  • 6 CSS3 Off-canvas navigation animations (push, reveal, scale up, rotate in, rotate out, rotate & reveal)
  • Resume section with clean typography
  • 4 types of charts (bar chart, pie chart, donut chart, progress chart) with on-scroll animations
  • AJAX portfolio section
  • 3 types of predefined layouts plus the option to use your own layout for portfolio details
  • Image slider and YouTube / Vimeo videos can be used in portfolio details
  • Book section with CSS3 animated books
  • 2 predefined 3D transitions for animated books plus 2D fallback
  • Gallery section with lightboxes for images, YouTube / Vimeo videos and webpages
  • Styled / original Google Maps
  • Working AJAX contact form
  • Lots of code snippets to help you create your own content

Getting Started

The Bootstrap 3 CSS stylesheets used in this template are linked directly to CDNs (Content Delivery networks). Before editing the template, please make sure that you are connected to the Internet. Otherwise, the template will not be able to work properly.

Installing the MongKok Theme in WordPress

We recommend installing the MongKok theme in a new WordPress site. If you want to install the theme on an existing WordPress site, please make sure that you back up all the important documents and files in your site before doing so.

To ensure that the MongKok theme can work smoothly, please install it on WordPress 3.9+.

To install the theme,

  1. Log on to your WordPress site and navigate to Appearance > Themes.
  2. Click the Add New button at the top and then click Upload Theme.
  3. Select the theme package (mongkok-wp-[version number].zip) and click Install Now.
  4. After the theme has been successfully installed, click Activate.
  5. Next, navigate to Tools > Import and click the WordPress link at the bottom.
  6. If you haven't installed the WordPress Importer, click Install Now to install it. If you already have the WordPress Importer installed, you can skip Step 7 and proceed to Step 8.
  7. Click Activate Plugin & Run Importer after the importer is installed.
  8. Select the MongKok quickstart XML file (mongkok-wp-quickstart.xml) from the theme package and and click Upload file and import.
  9. Next, you can assign the imported content to an author. Choose the option that meets your need (Very often, you want to assign posts to an existing user) and then check Download and import file attachments and click Submit.
  10. After the quickstart content has been successfully imported, navigate to Settings > Reading, and check A static page in the Front page displays field. Choose the MongKok Homepage in the drop-down list Front page and choose MongKok Blog in the drop-down list Posts page. Then click Save Changes.
  11. Now we will set up the menu. Navigate to Appearance > Menus and click Screen Options at the top.
  12. In the Show on screen section, check Sections and Links and uncheck other options. In the Show advanced menu properties section, check Description.
  13. Next, switch to the Manage Locations tab. Select the menu MongKok Navbar and assign it to the theme location MongKok Off-Canvas Navigation. Then click Save Changes.
  14. Navigate to Appearance > Widgets. Refer to the section Adding / Removing Sidebar Widgets in Blog to setup the recommended widgets.
  15. Finally, navigate to Posts and trash the default Hello world! post installed by WordPress. (You can restore the post later in case you need it.)
  16. Now you will have a working MongKok theme running on WordPress.

Changing the Color Scheme

A color scheme in the MongKok template consists of two colors: primary color and secondary color. To change the color scheme, navigate to MongKok > Color Scheme in WordPress' admin interface. In the Color scheme field, select Custom Color Scheme. Then change the primary color through the field Primary color and change the secondary color through the field Secondary color.

You can also use a single color instead of dual colors in the color scheme. Simply make the value of the field Secondary color equal to the value of the field Primary color. You will then have a monotone version of the template.

The MongKok template comes with 5 predefined color schemes. You can choose a scheme you like in the Color scheme field.

The Color Scheme tab also contains advanced color settings for other components of the theme. You can change and adjust their values according to your needs. However, most of the time the template will work well even if you don't change their values.

Changing the Typeface

3 different fonts are used in the body text, the title text and the signatures of the MongKok template respectively. To change the typeface, navigate to MongKok > Typography in WordPress' admin interface and specify the Google Fonts to use through the field Body Font, Title Font and Signature Font.

In the Typography tab, you can also specifies the font size and the line height of the body text, and the font weights for the body text and the title text.

Changing the Position of Navigation Bar

There are 4 navigation bar positions available in the MongKok template. You can place the navigation bar at the top, bottom, left or right of the browser viewport.

To change the position of the navigation bar, navigate to MongKok > Navigation Bar in WordPress' admin interface. Select Top, Bottom, Left or Right from the field Navigation Bar Position according to your preference and then save the changes you have made.

Changing the Off-canvas Navigation Animation

There are 6 off-canvas navigation animations available in the MongKok template. Most of the time, the navigation bar is hidden from the browser viewport, and it will move into the viewport once you click the navigation bar toggle. The 6 animations available are push, reveal, scale up, rotate in, rotate out and rotate & reveal.

To change the navigation animation, navigate to MongKok > Navigation Bar in WordPress' admin interface. Select Push, Reveal, Scale up, Rotate in, Rotate out or Rotate & Reveal from the field Off-Canvas Navigation Animation according to your preference and then save the changes you have made.

Adding Contents to the Navigation Bar

You can add your name, menu items, social links and other information to the navigation bar. To do so, navigate to MongKok > Navigation Bar in WordPress' admin interface. You can then add your name to the Content before Menu field, and your social links to the Content after Menu field. If you want to restore the default contents of these two fields, click the Reset Section button at the top.

If the navigation bar is at the top / bottom of the browser viewport, the contents before menu and after menu will be hidden. These contents will only be visible when the navigation bar is at the left / right of the browser viewport.

If you need a menu item to link to an external resource, navigate to Appearance > Menus in WordPress' admin interface. You can then add a link to the menu through the Links panel on the left. Once it is added to the menu, expand the link item in the section Menu Structure on the right. Through the field Description, you can assign a FontAwesome Icon to this link item. You can also drag the link item inside Menu Structure to rearrange its position. Finally, remember to save the changes by clicking the Save Menu button at the top.

Other Settings of the Navigation Bar

To access other settings of the navigation bar, navigate to MongKok > Navigation Bar in WordPress' admin interface. To change the colors used in the navigation bar, switch to the Color Scheme tab and change the related settings.

Creating / Removing a Section

To create a new section,

  1. Navigate to Sections > Add New Section in WordPress' admin interface.
  2. Give this section a title, which will appear in the navigation bar of the MongKok theme.
  3. Add content to this section by typing HTML codes through WordPress' editor. The code snippets page that comes with this theme package can help you add HTML codes that are inline with the MongKok theme style.
  4. Scroll to the bottm and select General as the template for this new section. (Ohter section templates are for the existing built-in MongKok sections.)
  5. You may also want to specify the heading for this section if it is different from the title.
  6. Give this section an order value through the Attributes panel on the right. Sections with smaller order values will appear first.
  7. Click Publish to save the new section.
  8. Next, navigate to Appearance > Menus in WordPress' admin interface.
  9. Select the newly created section in the Sections panel on the left and click Add to Menu.
  10. Expand the menu item for the new section in Menu Structure on the right. Specify the FontAwesome icon for this section through the Description field and click Save Menu. Now A menu item for the new section will be created in the navigation bar.

To remove a section, navigate to Sections in WordPress' admin interface. Hover over a section and click Trash. We do not recommend permanently deleting existing sections that come with the MongKok template since it will not be possible to restore them if you need them later. If you don't need to use a particular section, simply trash it.

Reordering Sections

Section order is determined by the order value specifed in the Attributes panel of each section. Sections with smaller order values will appear first in the front page.

Shortcodes in Sections

The MongKok WordPress theme uses the following shortcodes to render parts of the section contents. If some section contents do not show up, you may want to check if they have been accidently deleted.

Shortcode Syntax Functionality Remarks
[profile_pic] Render the profile picture
[list type="{Content Type}"] or
[list type="{Content Type}" half_margin_bottom=1]		 Render an item list (with half the margin-bottom value if specified) {Content Type} can be education, experience, or honor.
[chart id="{Chart ID}"] Render an animated chart {Chart ID} is required.
[contact_form] Render the contact form

Adding Contents to the Welcome Section

The welcome section is the first section of this template. This section is designed to display your basic information such as your name and your contact methods in a way that grabs attentions. The content of this section is defined in the following format.

<p class="before-name title-font uppercase letter-spacing">[Any text before name]</p>
<h1 class="name"><span class="span-text">[Your name]</span></h1>
<p class="after-name"><span class="span-text title-font">[Any text after name]</span></p>
<div class="row info">
	<div class="col-md-4"><p><i class="fa fa-mobile-phone"></i>[Telephone]</p></div>
	<div class="col-md-4"><p><i class="fa fa-envelope-o"></i><a href="#">[Email]</a></p></div>
	<div class="col-md-4"><p><i class="fa fa-laptop"></i><a href="#">[Website]</a></p></div>
</div>

There are basically 4 subsections in the welcome section: the before-name subsection, the name subsection, the after-name subsection and the info subsection. Any text that you want to appear before your name can be placed in the before-name subsection. Your name can be placed in the name subsection, and any text that you want to appear after your name, such as your position, can be placed in the after-name subsection. Your contact info, such as your telephone, email and website can be placed in the info subsection.

The class span-text is to make the text span the whole width of its container. The class title-font is to indicate the use of the font for the title text instead of the body text. The other classes used in the code above should be quite self-explanatory. For more about the use of the typography classes, please refer to the code snippets page that comes with the theme package.

Using Ticker

In the name subsection, besides using static text as in the example above, you can also use the following code to create a ticker.

<div class="ticker">
  <h1 class="name"><span class="span-text">I'm Creative</span></h1>
  <h1 class="name"><span class="span-text">I'm Talented</span></h1>
  <h1 class="name"><span class="span-text">I'm John Doe</span></h1>
</div>

Changing the Height of the Welcome Section

To change the height of the welcome section, navigate to MongKok > Welcome in WordPress' admin interface. Then change the value of the field Height of the Welcome Section to the height you want to use.

The unit of the height value can be px or %. Some typical height values are 100%, 80% and 500px.

Fix / Unfix the Welcome Section

You can fix / unfix the welcome section in the MongKok template. If the welcome section is fixed, it will stay at the same position when you scroll the webpage.

Navigate to MongKok > Welcome in WordPress' admin interface. Check or uncheck the field Fix the Welcome Section to fix / unfix the welcome section.

Showing / Hiding Box Shadow

To show / hide the box shadow at the bottom of the welcome section, navigate to MongKok > Welcome in WordPress' admin interface. Set the value of the field CSS Box Shadow to any valid value of the CSS property box-shadow such as 0 0 15px 0 rgba(0,0,0,.5) or none. Please refer to this webpage for details of the box-shadow property.

Other Settings of the Welcome Section

To access other settings of the welcome section, navigate to MongKok > Welcome in WordPress' admin interface. To change the colors used in the welcome section, switch to the Color Scheme tab and change the related settings.

Adding Contents to the About Section

The about section lets you add your profile picture and some descriptions about yourself.

To change the profile picture, navigate to MongKok > Profile Picture in WordPress' admin interface. Then select / upload your profile picture through the field Profile Picture.You can also select an image style for your profile picture from the field Image Style of Profile Picture.

This field Maximum Width of Profile Picture is to impose the maximum width for the profile picture, so that it will not appear too large on mobile devices. Its default value is 300px.

To change the intro part of the about section, navigate to Sections in WordPress' admin interface and click the About section. Edit its content to modify the introduction. You can use drop caps, bold text, links and other typographic styles. The HTML code snippets for these styles can be found in the code snippets page included in the theme package. The shortcode [profile_pic] is used to render your profile picture.

Adding Education and Experience to the Resume Section

Education and experience in the resume section are actually two item lists. Each education item and each experience item are defined in the following formats.

    <div class="row">
      <div class="col-sm-4">
        <p class="title-style sm-text-center">[University name]</p>
        <p class="tag sm-text-center">[Period]</p>
      </div>
      <div class="col-sm-8">
        <p class="title-style">[Degree earned]</p>
        <p>[Additional info]</p>
      </div>
    </div>
    <div class="row">
      <div class="col-sm-4">
        <p class="title-style sm-text-center">[Company name]</p>
        <p class="tag sm-text-center">[Period]</p>
      </div>
      <div class="col-sm-8">
        <p class="title-style">[Position]</p>
        <p>[Job description]</p>
      </div>
    </div>

To edit an education / experience item, navigate to Education or Experience in WordPress' admin interface. Select an item inside to edit it.

To create a new item, navigate to Education or Experience and select Add New Education Item and Add New Education Item in WordPress' admin interface. Use the HTML format above to create a new education / experience item.

To remove an education / experience item, navigate to Education or Experience in WordPress' admin interface. Simply mouse over an item and click Trash.

Adding Honors and Rewards to the Resume Section

Each item of honors and rewards is defined in the following format.

<p><span class="title-style">[Honor / Award name]</span>
      <br>[Additional info]</p>

To edit an honors and rewards item, navigate to Honors in WordPress' admin interface. Select an item inside to edit it.

To create a new item of honors and rewards, navigate to Honors and select Add New Honor Item in WordPress' admin interface. Use the HTML format above to create a new honor item.

To remove an item of honors and rewards, navigate to Honors in WordPress' admin interface. Simply mouse over an item and click Trash.

Adding Charts to the Resume Section

There are 4 types of charts available in the MongKok template: bar charts, donut charts, pie charts (a special case of donut charts), and the progress charts. We recommend creating your own charts based on the existing ones in the sample data. If you don't need a particular chart, you can simply trash it instead of permanently deleting it. By doing so, you can reuse the chart quickly by untrashing it in case you need it later. If you need more charts, you can create one more efficiently by referring to the sample charts.

To create a chart,

  • Navigate to Charts > Add New Chart in WordPress' admin interface.
  • Give this chart a title. You may also want to add some descriptions for this chart which will appear above the chart through WordPress' editor.
  • Scroll to the bottom and specify the chart ID. Please note that each chart requires a unique ID in order to work properly.
  • Specify the chart type and add chart items, item values and item colors.
  • Follow the tips on screen to specify other attributes of the new chart if necessary.
  • Click Publish to save and create the new chart.

HTML Code of Sample Charts

The HTML code of sample charts are available in the code snippets page that comes with the theme package. It is possible to create charts by copying the code snippets and paste them to WordPress' Editor when editing a section.

Other settings of Charts

To set up charts in the MongKok template, navigate to MongKok > Bar Chart, Donut / Pie Chart or Progress Chart in WordPress's admin interface. The settings there give you various options to fine-tune the animated charts.

Adding a Portfolio Category

To add a portfolio category,

  1. Navigate to Portfolio > Portfolio Category in WordPress's admin interface.
  2. Enter the name and the slug for the new category. Please note that portfolio categories are ordered by slug.
  3. Click Add New Portfolio Category to save the new category.

Adding Items to a Portoflio Category

To add a portfolio item,

  1. Navigate to Portfolio > Add New Portfolio Item in WordPress's admin interface.
  2. Type in the name of this portfolio item and its content through WordPress' editor.
  3. Scroll to the bottom of the page, you will see more options that allow you to further set up the portfolio item.
  4. Give this portfolio item a thumbnail through the field Thumbnail.
  5. Through the field Layout, you can select one of the three predefined layouts for portfolio items provided by the MongKok template: Image slider, YouTube Video and Vimeo Video. If you would like to use your own layout, select Free Layout.
    • If you use an image slider layout, you can then proceed to the Slider Images field.
    • If you use an YouTube video layout, you can then proceed to specifies the URL of the video through the field YouTube Video.
    • If you use an Vimeo video layout, you can then proceed to specifies the URL of the video through the field Vimeo Video.
  6. Through the field Sidebar Position, you can place the sidebar content on the left or on the right. If you don't need a sidebar, select No Sidebar.
    • If sidebar is used, you can then proceed to add content of the sidebar through the field Sidebar Content.
  7. Specify the portfolio categories you would like to add this item to through the Portfolio Categories panel on the right.
  8. Give this portfolio item an order value through the Attributes panel on the right. Sections with smaller order values will appear first.
  9. Click Publish to publish this portfolio item.

Since portfolio items are loaded using AJAX, when editing the HTML code of an item, we recommend you use absolute paths in all the a, img and iframe tags. If you want to use relative paths in these tags, please use:

  1. the data-href attribute instead of the href attribute in the a tags,
  2. the data-src attribute instead of the src attribute in the img tags and the iframe tags.

Otherwise, URLs of images and links will not be interpreted correctly.

Setting Up the Portfolio Section

To set up the portfolio section, navigate to MongKok > Portfolio in WordPress's admin interface. Most of the settings there are self-explanatory.

Editing the Events Section

To edit the description of the events section, navigate to Sections in WordPress's admin interface and click Events. Edit through WordPress' editor to modify the description.

To add an event,

  1. Navigate to Events > Add New Event in WordPress's admin interface.
  2. Give the new event a name and type in the content of this event through WordPress' editor.
  3. Scroll to the bottom of the page, and specify the date and location of the event.
  4. Give this event an order value through the Attributes panel on the right. Sections with smaller order values will appear first.
  5. Click Publish to publish this event.

Adding an Animated Book

To add a animated book,

  1. Navigate to Books > Add New Book in WordPress's admin interface.
  2. Type in the name of this book and its description through WordPress' editor.
  3. Scroll to the bottom of the page, you will see more options that allow you to further set up the book.
  4. Specify the background color for the back / front / left face of the book in Hex format.
  5. Give this book a cover image through the field Book Cover Image.
  6. Through the field Book Info (inside animated book), type in the HTML code of the content that will appear when the animated book is opened.
  7. Through the field Book Info (under description), type in the HTML code of the content that will appear below the description of the book.
  8. Give this book an order value through the Attributes panel on the right. Sections with smaller order values will appear first.
  9. Click Publish to publish this book.

Changing CSS3 Transitions of Animated Books

Using the Predefined Book Transitions

There are 2 predefined 3D transitions for animated books. To change the transition, navigate to MongKok > Animated Books in WordPress's admin interface. Choose an animation through the field CSS Book Animation.

Customizing Book Transitions

If you are familar with CSS3 transform, you can even try creating your own book transitions. Simply select Custom Animation from the drop-down list of CSS Book Animation. Then you can specify the inital and hover CSS transform for books and book covers through the fields Initial CSS Transform Of Animated Books, Hover CSS Transform Of Animated Books, Initial CSS Transform Of Book Covers Of Animated Books and Hover CSS Transform Of Book Covers Of Animated Books.

If you are not familiar with CSS3 transform, it is highly recommended that you use one of the predefined book transitions. Since the stability of new book transitions is not guaranteed.

Other Settings of Animated Books

To set up the books section, navigate to MongKok > Animated Books in WordPress's admin interface. To change the colors used in the 2D fallback of animated books, switch to the Color Scheme tab and change the related settings.

Setting up the Blog Section

To set up the blog section, navigate to MongKok > Blog in WordPress's admin interface. You can then specify whether or not to link the blog menu item in the navbar to the blog index page on blog pages, the number of blog posts shown in the blog section of the homepage, and the content of blog footer etc. Most of the settings are self-explanatory.

To add a widget on the right sidebar, navigate to Appearance > Widgets in WordPress's admin interface. Then you can drag widgets from Available Widgets and drop them to Blog right sidebar to add them. To remove a widget on the right sidebar, expand an existing widget from Blog right sidebar and click Delete.

Adding / Removing Sidebar Widgets in Blog

To add a widget to the right sidebar, navigate to Appearance > Widgets in WordPress's admin interface. Then you can drag widgets from Available Widgets and drop them to Blog right sidebar to add them. To remove a widget from the right sidebar, expand an existing widget from Blog right sidebar and click Delete. We recommend removing Search from the right sidebar, since there is already a search bar displayed at the top of each blog page.

We recommend adding Recent Posts, Archives, Categories, Tag Cloud and Recent Comments to the right sidebar, with the option Display post date? enabled in Recent Posts and the option Show post counts enabled in Archives.

Adding / Removing a Blog Post

To add a blog post,

  1. Navigate to Posts > Add New in WordPress's admin interface.
  2. Type in the title of this blog post and its content through WordPress' editor.
  3. Use the More button in WordPress' Editor to define the boundary of the excerpt of this blog post. The excerpt will be displayed in the blog index page.
  4. Scroll to the bottom of the page, you will see more options that allow you to further set up the blog post.
  5. Specifies the categories and tags of this blog post through the fields Categories and Tags on the right sidebar.
  6. Give this blog post item a thumbnail through the field Featured Image on the right sidebar.
  7. Through the field Homepage Summary, give this blog post a summary which will be displayed in the blog section on the homepage. The homepage summary can be different from the excerpt defined in Step 3 and is a required field.
  8. Click Publish to publish this portfolio item.

To remove a blog post, navigate to Posts in WordPress' admin interface. Hover over a blog post and click Trash.

Approving / Deleting a Comment

By default, a new comment needs to be approved before it will be displayed on a blog post.

To approve a comment,

  1. Navigate to Comments in WordPress's admin interface.
  2. Hover over a comment and click Approve.

To delete a comment,

  1. Navigate to Comments in WordPress's admin interface.
  2. Hover over a comment and click Trash.

Adding Items to the Gallery Section

The gallery section allows you to show your images, YouTube / Vimeo videos and webpages in a lightbox. The lightbox feature is provided by the Colorbox plug-in.

To add a gallery item,

  1. Navigate to Lightboxes > Add New Lightbox in WordPress's admin interface.
  2. Type in the title of this lightbox.
  3. Scroll to the bottom of the page, you will see more options that allow you to further set up the lightbox.
  4. Through the field Thumbnail, specify the thumbnail for this lightbox.
  5. Choose a content type for the lightbox. The 4 available content types are Image, YouTube Video, Vimeo Video and External Webpage.
  6. Depending on the content type, specify the URL of the image, the YouTube Video, the Vimeo Video or the external webpage through the respective field.
  7. Give this lightbox an order value through the Attributes panel on the right. Sections with smaller order values will appear first.
  8. Click Publish to publish this lightbox.

Lightboxes outside the Gallery Section

You may also use the lightbox feature outside the gallery section. Please refer to the code snippets page for the HTML code format.

Setting Up the Gallery Section and Lightboxes

To set up the gallery section, navigate to MongKok > Gallery in WordPress's admin interface. To set up lightboxes, switch to the Lightboxes tab.

Setting up Google Maps

To set up Google Maps, navigate to MongKok > Google Maps in WordPress's admin interface. You can specify the latitude, longitude and zoom level of the map. Also you can change the appearance of the map by changing the fields of Gamma, Saturation, Lightness and Invert Lightness.

Setting up the Contact Form

To set up the Contact Form, navigate to MongKok > Contact Form in WordPress's admin interface. Type in the email address in the Contact email field to receive messages submitted through the contact form. You can also customize the messages about submission status through the fields Success Message and Failure Message.

The contact form in this template is powered by ajax contact form written by jigowatt.

When you navigate to Pages in WordPress's admin interface, you will see a page titled MongKok Contact Form Processor. Please make sure that the template for this page is also set to MongKok Contact Form Processor and do not modify or remove this page as it is necessary and related to the functionality of the contact form.

Editing contact information and social links

To edit contact information, navigate to Sections in WordPress's admin interface and select Contact. You can then edit various contact information such as address, telephone, email address and web address through WordPress' editor.

To edit social links, please add / remove social links according to your needs inside the ul tag of the class name social.

Please do not delete the short code [contact_form] as it is required in order for the contact form to work properly.

Setting up the Footer

To edit the footer, navigate to Sections in WordPress's admin interface and select Footer to edit its content.

You can link to your PDF resume through the Download Resume button, add your own signature and modify the copyright information in the footer section.

To change the colors used in the footer, switch to the Color Scheme tab and change the related settings.

Setting up Google Analytics

To set up Google Analytics, navigate to Sections > General in WordPress's admin interface. Copy and paste the Google Analytics code to the field Google Analytics.

Search Engine Optimization (SEO)

To define meta keywords and meta description for search engines, navigate to Sections > General in WordPress's admin interface. Enter a comman-separated list of keywords to the field Meta Keywords. Enter a short and concise description of your website through the field Meta Description.

Adding Custom CSS

If you are familiar with CSS, you can write your own CSS to overwrite the original template styling to further customize the MongKok theme.

To add your own custom CSS, navigate to Sections > General in WordPress's admin interface. Enter your own CSS without the <style> tag to the field Custom CSS.

Browser Compatibility

This template supports Chrome, Firefox, Safari and IE9+.

Note that some features of this template may be disabled on certain devices.

  • Off-Canvas navigation animations require the support of CSS3 transitions and animations, and therefore are not available in IE9, where a jQuery fallback is provided.
  • 3D CSS3 animated books require the support of the CSS3 property transform-style, and therefore are not available in IE9, IE10 and IE11, where a 2D fallback is provided.
